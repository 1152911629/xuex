## 特色功能
1. 在线脚本：每次启动会在线请求最新的UI和脚本，实现热更新（可能带来安全及隐私问题，但有开源项目作为监督，总体利大于弊）
2. 采用多个在线题库，优化答题速度与准确率
3. 可选百度OCR或本地PaddleOCR帮助完成四人赛和双人对战，具有较高的准确率
